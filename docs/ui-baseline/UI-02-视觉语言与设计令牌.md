# 视觉语言与设计令牌

## 总体气质

当前 UI 的视觉方向不是严肃金融盘，而是“可爱、轻盈、糖果感、带一点舞台气氛”的链上工具界面。

核心关键词：

- Bubble
- Candy
- Soft Glass
- Rounded
- Playful but readable

## 字体系统

当前 `:root` 中的字体令牌分为多组用途：

- 展示字：`Baloo 2`
- 正文字：`Noto Sans SC`、`Nunito`
- 品牌字：`Baloo 2`
- 叙事字：`ZCOOL XiaoWei`，中文场景下替换为 `M PLUS Rounded 1c`
- 数字字：`Nunito`

这意味着当前界面并不是单字体方案，而是按内容类型混排。

## 色彩系统

当前主色不是单一品牌色，而是多组低饱和糖果色共同构成氛围：

- `--peach`
- `--mint`
- `--butter`
- `--sky`
- `--lilac`
- `--rose`

基础背景是暖白到浅粉、浅紫、浅绿的长渐变，不是纯白底。

## 背景与氛围

当前背景层由多个元素叠加：

- 页面整体渐变底色
- `aurora` 模糊光团
- `bubble-field` 动态气泡
- `grid-haze` 轻雾网格
- pointer 跟随高光

因此页面视觉重心并不只来自卡片本身，而是整页的环境氛围。

## 卡片语言

当前几乎所有核心模块都采用圆角软卡片系统：

- 大圆角
- 半透明浅色底
- 内描边
- 柔和阴影
- 轻微 hover 抬升

这套卡片语言同时用于：

- 指标卡
- 借款面板
- 借款订单卡
- FAQ 卡
- 开发者面板卡片

## 按钮语言

当前按钮有三层：

- `primary-button`
- `secondary-button`
- `ghost-button`

主按钮带更强的色块感和高光效果，次按钮与幽灵按钮则偏轻。

## 数字与数据样式

数字使用单独的数据字体栈，并强调：

- `tabular-nums`
- `lining-nums`
- 大字号主数值
- 紧凑的字距

当前价格位还支持超小数下标展示，这是当前界面里最有辨识度的细节之一。

## 动效语言

当前页面的动效是“轻快但不硬核”，主要包括：

- 区块 reveal
- 卡片轻微 tilt
- 指标数字补间动画
- 借款估值刷新动画
- toast 显隐
- 装饰层漂浮与高光

整体是增强“活着”的感觉，而不是做强烈的炫技动画。
